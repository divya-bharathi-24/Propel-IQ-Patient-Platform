# User Story - US_016

## Story ID

- ID Format: US_016

## Story Title

- Patient Dashboard Aggregation

## Description

- As an authenticated Patient, I want a personalized dashboard showing my upcoming appointments with status, pending intake items, document upload history, and 360-degree view availability indicator, so that I can manage all my healthcare interactions from a single view.

## Acceptance Criteria

1. **Given** I am authenticated and navigate to my dashboard, **When** the page loads, **Then** I see a list of my upcoming appointments with date, time, specialty, and current status (Booked/Arrived/Completed/Cancelled).

2. **Given** I have a pending intake form for a booked appointment, **When** I view my dashboard, **Then** the pending intake item is shown with a "Complete Intake" call-to-action linked to the correct appointment.

3. **Given** I have previously uploaded clinical documents, **When** I view the dashboard, **Then** the document upload history section shows file names, upload dates, and processing statuses (Pending/Processing/Completed/Failed).

4. **Given** my 360-degree patient view has been verified by staff, **When** I view the dashboard, **Then** a "360° View Ready" indicator is displayed; if not yet verified, the indicator shows "Pending Staff Verification".

## Edge Cases

- What happens when I have no upcoming appointments? (Dashboard shows an empty state with a "Book Appointment" call-to-action)
- What happens when a document upload fails processing? (Dashboard shows "Processing Failed" status with a retry upload option)
