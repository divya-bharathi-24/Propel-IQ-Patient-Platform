# User Story - US_044

## Story ID

- ID Format: US_044

## Story Title

- Data Conflict Detection, Visual Highlighting & Resolution

## Description

- As an authenticated Staff member, I want AI-detected conflicts across a patient's uploaded documents (conflicting medications, dosages, allergies, diagnoses) to be visually highlighted with severity classification, so that I can resolve each conflict before the patient's 360-degree profile is marked Complete.

## Acceptance Criteria

1. **Given** the AI extraction pipeline detects a conflicting data element across two documents, **When** the DataConflict record is created, **Then** it includes `fieldName`, `value1`, `sourceDocumentId1`, `value2`, `sourceDocumentId2`, `resolutionStatus = Unresolved`, and severity classification (`Critical` or `Warning`).

2. **Given** conflicts exist in the 360-degree view, **When** I view the affected fields, **Then** each conflict is displayed with a visually distinct indicator (red border for Critical, amber for Warning), showing both conflicting values side by side with the source document identified for each.

3. **Given** I review a conflict and select the authoritative value, **When** I submit the resolution, **Then** the DataConflict record is updated with `resolutionStatus = Resolved`, `resolvedValue`, `resolvedBy = staffId`, and `resolvedAt = UTC timestamp`.

4. **Given** all Critical conflicts are unresolved, **When** I attempt to mark the 360-degree profile as "Complete", **Then** the action is blocked with a message listing each unresolved Critical conflict that must be resolved first.

## Edge Cases

- What happens when I disagree with both conflicting values and need to enter a third authoritative value? (The resolution interface allows free-text entry for a manually specified authoritative value alongside the two AI-detected options)
- What happens when a new document is uploaded after conflicts are resolved? (New extraction triggers a re-run of conflict detection; newly detected conflicts are presented as unresolved; previously resolved conflicts are retained)
