# User Story - US_049

## Story ID

- ID Format: US_049

## Story Title

- AI Safety Guardrails & Immutable Prompt Audit Logging

## Description

- As a system operator, I want all AI prompts to have PII redacted before transmission to external providers, document-level access control enforced in retrieval pipelines, harmful content filtered from outputs, and all AI prompts/responses immutably logged for 7 years, so that the platform is HIPAA-compliant and patient data is never exposed to unauthorized AI providers.

## Acceptance Criteria

1. **Given** an AI prompt is constructed for an external provider (OpenAI), **When** the PII redaction middleware runs, **Then** all patient identifiers (name, DOB, email, phone, address, insurance ID) are replaced with anonymized tokens before the prompt is transmitted.

2. **Given** a RAG pipeline query runs, **When** document chunks are retrieved, **Then** the ACL filter verifies the requesting user is authorized to access each retrieved document's patient context; unauthorized documents are excluded from the prompt context.

3. **Given** an AI model returns a response, **When** the content filter middleware evaluates the output, **Then** responses containing harmful, biased, or clinically inappropriate content are blocked, an alert is logged, and the request falls back to manual review.

4. **Given** any AI prompt or response is generated, **When** the interaction completes, **Then** an immutable AuditLog record is created containing the prompt (with PII already redacted), the response, model version, timestamp, and requesting user ID — retained for 7 years.

## Edge Cases

- What happens if the PII redaction service fails? (The AI request is blocked entirely until redaction succeeds; no prompt is sent to external providers with un-redacted PII)
- What happens when the content filter incorrectly blocks a valid clinical response (false positive)? (The block is logged; the staff member is shown a "Review blocked by content filter" message and can request a manual review escalation)
